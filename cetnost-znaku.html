Četnost znaků

Post in Út 05 květen 2015 |Tagy: #mírně pokročilý #slovník

Vytvořte program, který ze standardního vstupu nebo ze souboru načte text a vypíše četnosti jednotlivých znaků textu na obrazovku. Četnosti znaků zobrazte nejen jako číselný údaj ale i graficky pomocí jednoduchého horizontálního bargrafu. Až se vám bargraf povede udělat horizontálně udělejte ho vertikálně.

Malá pomoc:

from sys import stdin, stdout, stderr

cetnosti = {}

while 1:
    line = stdin.readline()
    if not line:
        break
    line = line.decode('utf-8')
    line = line.strip()
    ...
    ...

# tiskne seřazené četnosti
for znak in sorted(cetnosti.keys()):
    print znak,'->',cetnosti[znak]

# poměrně neefektvní zjištění maximální četnosti znaku
hodnoty = cetnosti.values()  # všechny hodnoty ve slovníku
hodnoty.sort()               # seřadím je
hodnoty.reverse()            # nejčetnější je na začátku
print "nejčetnější znak je zastoupen", hodnoty[0] , "krát"

print 20*"*"
